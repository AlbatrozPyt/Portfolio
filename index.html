Sobre o projeto

Durante meu curso técnico, fomos instruidos pelos professores a criarmos um projeto mobile, sobre um tema livre, em grupo.

Eu e meus colegas idealizamos a plataforma Voyager, a Voyager é uma plataforma criada para te ajudar a fazer sua viagem, gerenciando sua viagem através de checklists de tarefas e a opção de visualizar e criar postagens sobre as suas últimas viagens. Além da opção de utilizar a IA generativa da Google, o Gemini, em um chatbot voltado para te ajudar com dicas, e respondendo todas as suas dúvidas.

Tecnologias

    Front-End

  • Figma (prototipação).

  • React Native.

    • Back-End

  • A API foi feita com C#, utilizando o framework ASP.NET Web Api.

  • O banco de dados foi gerenciado na Azure, onde a API foi hospedada, também foi utilizado o serviço de Blob Storage para armazenar imagens.

    • IA

  • Foi utilizado a IA generativa da Google, o Gemini, para a criação de um chatbot, e também para fazer buscas sobre lugares que o usuário pesquisar.
